# EchoSpace Skills

Agent skills for working with the [`.echo` conversation format](https://github.com/stonexer/echospace) — the open-source, local-first prompt debugging workspace.

## Sub-skills

### echospace-export

Convert conversation files from OpenAI, Anthropic, Google, Vercel/Helicone, or raw text into `.echo` format.

### echospace-integrate

Integrate `.echo` export into your own app so users can send conversations to EchoSpace for review and replay.
